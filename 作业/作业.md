2. 从键盘上输入圆的半径，计算圆的体积，运行结果如下面所示。 

- 输入的半径是字符串，需要转换为数值 
- π的值取3.14就可以了了 

```

```

```python

PI = 3.14
area = PI*radius*radius

```
