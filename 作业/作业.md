8.输出摄氏温度---华氏温度对照表，摄氏温度从0~100，每隔5度显示一个值。

- 提示：C＝5/9（F－32），C表示摄氏温度，F表示华氏温度

~~~
摄氏温度	华氏温度
0			32
5			41
10			50
15			59
…
~~~

```python
C = 0
print('摄氏温度 华氏温度')
for C in range(0,101,5):
    F = (9/5) * C + 32
    print('{0:^8}{1:^8}\t' .format(C,F))
```
