3. 输⼊年，写代码判断输⼊的年是否是闰年，并且打印对应的结果 。(是闰年的条件: 能被4整除但是 不能被100整除或者能够被400整除的年)

例如：输⼊ 2020 打印 True, 输⼊ 2011 打印 False

```python
value = input('请输入年：')
year = int(value)
if year % 100 == 0:
    if year % 400 == 0:
        print(value,'是闰年')
    else:
        print(value,'不是闰年')
else:
    if year % 4 == 0:
        print(value, '是闰年')
    else:
        print(value, '不是闰年')
```

例如: 输⼊体重: 55, 输⼊身⾼：1.55, 输出: True

```python

weight = float(input('请输入你的体重(公斤):'))

    print('身材正常')

    print('身材不正常')
```
